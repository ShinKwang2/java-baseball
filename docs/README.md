## 기능 목록

### 컴퓨터(상대) 관련
* 컴퓨터의 중복되지 않는 숫자 만들기[ ]
* 컴퓨터의 숫자 테스트하기[ ]

### 유저 관련
* 입력
    * 게임 시작 문구 출력 [ ]
    * 숫자 입력 기능 [ ]
        * 잘못된 값을 입력할 때 Exception 구현 [ ]
            * 중복된 숫자 검증[ ]
            * 3을 제외한 자리 수 검증[ ]
            * 숫자가 아닌 문자 입력 검증[ ]
* 출력
    * 입력한 숫자에 따라 결과를 출력
        * 숫자 값 + 순서가 같을 때 "스트라이크" 개수 [ ]
        * 숫자 값 + 순서가 다를 때 "볼" 개수 [ ]
        * 아무 것도 일치하지 않을 때 "낫싱" [ ]
